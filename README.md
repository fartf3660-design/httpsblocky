The Gemini - Block Coding Studio is a lightweight, professional-grade web development environment designed to bridge the gap between visual "Blueprint" logic and raw HTML coding.  

Here is a brief breakdown of how it works:

Modular Architecture: Instead of traditional text coding, users build websites by dragging "blocks" (Logic, Content, and Style) onto a dynamic workspace.  
Parent-Child "Main Block" Logic: The "Main Container" acts as the foundation; dragging it moves all nested blocks together, while pulling a block out of the container automatically removes it from the source code.  
Live Source Engine: A real-time viewer translates the physical layout of blocks into clean, structured HTML/CSS code instantly as you edit.  
Pro Toolset: The interface includes advanced features like a color wheel for background styling, a local file uploader for images, and a dedicated trash system for quick cleanup.  
Simplified Export: Once the design is finished, a single click "bakes" the project into a standard .html file that can be hosted or shared anywhere.
